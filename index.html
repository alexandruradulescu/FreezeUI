FreezeUI

Lightweight, vanilla javascript, micro plugin for freezing/blocking the UI.
No jQuery needed.

Documentation and Download Roadmap

Documentation

Download

Go to the GitHub page and download the version which fits you best (ES5 or ES6, full code or minified).

GitHub

Call the script

Link the downloaded script in the head or at the end of the body as you like

Freeze the full UI

Simply call the FreezeUI() function similar to the example below.

Unfreeze the UI

Simply call the UnFreezeUI() function similar to the example below.

Freeze only a piece of the UI

Call the FreezeUI({selector: '.class-name'}) function while passing an (options) object with the desired selector.

Custom loading text

Call the FreezeUI({text: 'Magic is happening!'}) function while passing an (options) object with the desired text.
